# 🚀 TechVision - Premium SaaS Landing Page

A modern and responsive SaaS landing page built with Next.js, React, TypeScript, Tailwind CSS, Framer Motion, and GSAP.

---

## ✨ Features

### 🎨 Design & User Experience
- Modern and professional UI
- Dark & Light mode support
- Fully responsive design
- Smooth animations and transitions
- SEO optimized
- Accessibility friendly

### 📱 Website Sections
- Header Navigation
- Hero Section
- Features Section
- Product Showcase
- Team/About Section
- Achievements Section
- Statistics Section
- Testimonials Section
- Call-to-Action Section
- Footer

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

```bash
# Clone repository
git clone https://github.com/your-username/techvision-project.git

# Navigate to project
cd techvision-project

# Install dependencies
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

## 🎨 Customization

### Change Company Name

Update the following files:

```text
components/header.tsx
components/footer.tsx
app/layout.tsx
```

### Change Theme Colors

Edit:

```text
app/globals.css
```
